## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/riituv/portfolio.git
cd portfolio
```

### Install dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env.local` file in the root directory:

This project uses Resend to handle contact form submissions.
Sign up at https://resend.com and generate an API key.

```env
RESEND_API_KEY=your_resend_api_key
```

### Run the development server

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

---

## 🚀 Deployment

This project is optimized for deployment on Vercel.

```bash
npm run build
```

Push your code to GitHub and connect the repository to Vercel for automatic deployments.
